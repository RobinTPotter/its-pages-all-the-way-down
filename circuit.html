Help & Instructions

Basic Controls

  • Add Component (C): Press C or click New Component to open the component modal. Defaults are orientation vertical and size 1,3.
  • Select Component: Click on any component body to select it (it becomes the "selected" component).
  • Rotate (R): With a component selected, press R to rotate it 90° clockwise.
  • Duplicate (D): With a single component selected, press D to copy it with an offset position. With a lasso group selected, D duplicates the whole group (components, jumpers, and cuts) and selects the copies.
  • Delete Component: Hold Ctrl and click a component body.
  • Drag Component: Click and drag a component to move all its pins together. Drag preserves cursor offset (no jump on drag start).
  • Lasso Select: Click and drag on an empty area of the board to draw a selection box. Any components, jumpers, and cuts touched by the box are selected (highlighted in blue).
  • Move Group: With items selected, click and drag any selected item to move the whole group together.
  • Duplicate Group (D): With items selected, press D to duplicate the whole group; the new copies become the active selection.
  • Delete Group: With items selected, press Delete or Backspace to remove them all.
  • Pan & Zoom View: Hold the middle mouse button and drag to pan the board left/right/up/down. Use the mouse wheel to zoom in and out.
  • Close Modals: Press Escape to close open modals.

Jumpers & Connections

  • Add Jumper: Hold Shift and click a hole to start a jumper, then click another hole to complete it.
  • Move Jumper End: Drag the green handles on jumper endpoints to move them.
  • Delete Jumper: Hold Ctrl and click on a jumper to delete it.

Cuts & Strip Isolation

  • Add/Remove Cut: Hold Ctrl and click to toggle a cut segment. Cuts snap to strip segments (between adjacent holes), using segment-based X snapping.
  • Add/Remove Drill-hole Cut: Enable Drill-hole cut mode, then hold Ctrl and click a hole to isolate that hole from the strip.
  • Move Cut: Drag an existing cut marker to move it to another segment.
  • Move Multiple Cuts (Alt+Drag): Hold Alt while dragging a cut to move all cuts in the same row together by the same offset.

Board Editing

  • Edit Data (JSON): Click Edit Data to view and manually edit the board JSON. The editor will auto-select the last interacted component block when available.
  • Toggle Connectivity: Click Toggle Connectivity to highlight all connected regions on the board (helpful for debugging circuits).
  • Label by Net: Check this box to display pin net names instead of pin IDs.
  • Cuts on Top: Check this box to render cut marks above all other elements.
  • Clear Board: Click Clear Board, then confirm in the modal to remove all components, pins, cuts, and jumpers.

Board Object Model

The board data is one JSON object with four top-level collections:

  • components[]: logical parts and per-part metadata.
  • pins { [componentId]: { [pinId]: {row,col} } }: physical pin locations on the board grid.
  • cuts[]: strip break segments as {row, col} where col is the segment between hole col and col+1.
  • drillCuts[]: drilled-hole breaks as {row, col}, isolating that hole from adjacent strip copper.
  • jumpers[]: wires connecting two holes as { from:{row,col}, to:{row,col} }.

Component properties currently used:

  • id: unique component name (for example R1, U1).
  • pins[]: pin definitions with id and optional net.
  • canRotate: if false, the R rotate shortcut is ignored.
  • orientation, splitSides: creation/layout metadata.
  • footprint: {width,height}: body-size metadata used by rendering (does not move pin coordinates).
  • color: body fill color (hex).
  • opacity: body opacity from 0 to 1.

Minimal example:

{
  "components": [
    {
      "id": "R1",
      "canRotate": true,
      "orientation": "vertical",
      "splitSides": false,
      "footprint": { "width": 1, "height": 3 },
      "color": "#add8e6",
      "opacity": 0.8,
      "pins": [
        { "id": "1", "net": "VCC" },
        { "id": "2", "net": "N1" }
      ]
    }
  ],
  "pins": {
    "R1": {
      "1": { "row": 4, "col": 8 },
      "2": { "row": 6, "col": 8 }
    }
  },
  "cuts": [ { "row": 4, "col": 8 } ],
  "drillCuts": [ { "row": 4, "col": 9 } ],
  "jumpers": [ { "from": {"row": 0, "col": 0}, "to": {"row": 2, "col": 0} } ]
}

Example Workflow: Build a Simple LED Circuit

  1. Add a Battery: Press C, enter ID BAT1, set size 1,3, labels +,-, nets VCC,GND. Click Add.
  2. Add a Resistor: Press C, enter ID R1, set size 1,3, labels 1,2, nets VCC,N1. Click Add.
  3. Add an LED: Press C, enter ID LED1, size 1,2, labels A,C (anode/cathode), nets N1,GND, pick a red color. Click Add.
  4. Place Components: Drag each component to suitable positions on the board.
  5. Add Jumpers: Hold Shift and click holes to connect: BAT1(+) → R1(1), R1(2) → LED1(A), LED1(C) → BAT1(-).
  6. Optional Isolation: Add cuts with Ctrl+click where you need to break strips.
  7. Review: Click Toggle Connectivity to verify the intended net paths.
  8. Edit if Needed: Click Edit Data to tweak JSON values directly.
